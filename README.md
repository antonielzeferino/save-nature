# Salve a Natureza - Aplicação Web

Este é um projeto de página web interativa com foco na preservação da natureza. A página fornece informações e inspirações para práticas sustentáveis, com uma interface moderna e visualmente atraente.

## Funcionalidades

- **Banner Principal:** Uma imagem de fundo com texto sobre a importância de salvar a natureza.
- **Folhas Animadas:** Uma camada visual que adiciona um efeito natural ao fundo da página.
- **Seção Informativa:** Um parágrafo explicativo sobre a preservação da natureza.
- **Lista de Ações Ecológicas:** Inclui uma lista de ações práticas que os usuários podem realizar para preservar o meio ambiente.
- **Projetos Ecológicos:** Uma seção com projetos de preservação da natureza.
- **Rodapé:** Informações de contato e links para redes sociais.

## Tecnologias Utilizadas

- **Next.js:** Framework para renderização de páginas em React.
- **Tailwind CSS:** Biblioteca de classes utilitárias CSS para estilização rápida.
- **Imagem Responsiva (Next/Image):** Para otimizar o carregamento e exibição das imagens.
- **Componentização:** Uso de componentes React como `EcoFriendlyActionsList` e `EcoProjects` para modularizar o código.

## Como Executar o Projeto

1. Clone este repositório:

   ```bash
   git clone https://github.com/seu-usuario/salve-a-natureza.git
   ```

2. Instale as dependências:

   ```bash
   npm install
   ```

3. Execute o projeto:

   ```bash
   npm run dev
   ```

4. Acesse o projeto no navegador em `http://localhost:3000`.

## Imagens Utilizadas

As imagens de fundo são carregadas no diretório `public/backgrounds` e incluem:

- `main-bg.png` - Usada como imagem principal no banner de abertura.
- `noon.png` - Imagem de fundo para a seção de ações ecológicas.
- `sunset.png` - Imagem de fundo para a seção de projetos ecológicos.
- `night.png` - Imagem de fundo no rodapé.
- `leefs.png` - Folhas que decoram o banner principal.

## Estilo e Design

O projeto utiliza Tailwind CSS para garantir uma interface responsiva e estilizada com classes utilitárias, proporcionando um desenvolvimento rápido e fácil personalização dos estilos.
